# Easy Game - Frontend

A minimal React + Vite frontend using Tailwind CSS.

Setup:
1. `cd frontend`
2. `npm install`
3. `npm run dev`

Environment:
- Set `VITE_API_URL` to your backend, e.g. `http://localhost:4000/api`.

Design notes:
- Mobile-first layout with Tailwind utilities.
- Components: OddsList, BettingForm, UserPanel.
